# 사이트 번역 - 영어

웹사이트 서핑중에 영어로된 웹툰을 보게되었는데
웹툰도 보면서 공부도 같이 할 수 있었으면 좋겠다는 마음에서 만들게 되었습니다.

Used
- Google MlKit(OCR) : 이미지에서 텍스트 추출
- Firebase RemoteConfig(버전 관리 목적), Crashlytics(앱 오류 확인 목적)
- Retrofit2 & OkHttp3
- Room
- DataBinding & ViewBinding
- ViewModel & LiveData
- Koin
- Coroutine
